# WTShell
**Описание**
Генератор рукописного reverse-shell с использованием стеганографии.
Stager - скачивает с указанного url картинку, формирует из неё payload и запускает его.
reverse-shell - подключается к c2 серверу. Команды выполняются через powershell через base64. Соединение через tls.

Данный проект написан в учебных целях. Темы, которые присутсвуют: стеганография, обфускация, lolbas, XOR, WINAPI

Использование:
 .\server.py - для запуска c2 сервера на своих интерфейсах. По дефолту используется порт 443. Можно изменить внутри файла.

 .\create_binary.py - создаёт stager. r_host - ip-адрес c2 сервера, r_port - ip-адрес c2 сервера, url - ссылка на файл(картинку, где зашифрован payload). Если не указывать url, то в процессе будет создана картинка с вредоносным кодом, вы должны разместить её и указать url в процессе работы скрипта.
python create_binary.py output.exe [r_host] [r_port] [url]
